# 中文组会提纲模板

```markdown
# 组会汇报提纲

## 1. 开场一句话
- 本周主线：
- 最需要大家知道的变化：

## 2. 关键证据
- 文献证据：
- 实验证据：
- 项目决策或方向变化：

## 3. 当前判断
- 已确认：
- 初步但待验证：
- 暂时否掉 / 暂缓：

## 4. 当前最大卡点
- 卡点：
- 为什么卡住：
- 已经试过什么：

## 5. 下周交付物
- Deliverable 1：
- Deliverable 2：

## 6. 希望讨论的问题
- 最需要反馈的问题：
- 可选讨论点：
```

## 使用提醒

- 提纲应该让口头汇报更顺，而不是复制周报全文。
- 把“需要导师反馈”的问题前置，不要藏在最后一堆细节里。
- 每个部分用短句，适合直接照着讲。
